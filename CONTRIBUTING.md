# Contributing

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push to the branch (`git push origin feature/my-feature`)
5. Open a Pull Request

## Development Setup

```bash
git clone https://github.com/pancheng/chinese-sensitive-words-mcp.git
cd chinese-sensitive-words-mcp
npm install
npm run dev
```

## Reporting Issues

Please use [GitHub Issues](https://github.com/pancheng/chinese-sensitive-words-mcp/issues) to report bugs or request features.
